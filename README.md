# 🎧 Wavely

Wavely is a music app built to explore new sounds, create vibes, and make listening simple and fun.

🚀 Built by a 13-year-old developer passionate about tech and creativity.

---

## 🌐 Live App
👉 https://wavely-music.lovable.app

---

## ✨ Features
- 🎵 Stream and explore music
- 🔍 Discover new tracks and artists
- ⚡ Fast and simple interface
- 📱 Works in your browser
